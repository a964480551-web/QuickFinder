# QuickFinder 2.1.0

这是首个适合公开开源发布的版本。

## 主要变化

- 索引改为安全的 SQLite 格式。
- 应用不联网、不上传文件名、路径或内容。
- 安装程序改用开源 NSIS，并以当前用户权限安装。
- Qt 动态库使用目录式分发，附带第三方许可和对应源代码说明。
- 搜索、索引构建和缓存加载均在后台运行。

## 安装

下载 `QuickFinder_Setup_2.1.0.exe` 后运行。安装程序暂未使用商业代码签名证书，因此 Windows SmartScreen 可能显示“未知发布者”。请核对同一发布页中的 `SHA256SUMS.txt`。

## 隐私

索引仅保存在 `%LOCALAPPDATA%\QuickFinder`，不会通过网络发送。
