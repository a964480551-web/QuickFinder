# Qt / PySide6 对应源代码说明

本项目发布包使用未修改的 PySide6 6.11.1、Shiboken6 6.11.1 和 Qt 6.11.1 动态库，并按 LGPLv3 条款使用。

对应源代码可从 Qt 官方渠道获取：

- Qt for Python：https://code.qt.io/cgit/pyside/pyside-setup.git/tag/?h=v6.11.1
- Qt 6.11.1 源代码：https://download.qt.io/official_releases/qt/6.11/6.11.1/submodules/
- Qt 开源许可说明：https://www.qt.io/licensing/open-source-lgpl-obligations

项目没有修改上述第三方库。安装目录中的 Qt 动态库以独立文件分发，用户可以调试、检查或替换兼容版本。若官方链接失效，可通过 GitHub Issue 请求与发布版本相对应的第三方源代码定位信息。
